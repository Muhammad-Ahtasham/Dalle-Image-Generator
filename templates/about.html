About This App

This is a DALL路E-powered image generator web app. Enter a prompt to generate AI images, view your creations in the gallery, and download them for your use.

Technologies Used

  • Python 3
  • Flask (web framework)
  • OpenAI API (DALL路E 3)
  • HTML, CSS, JavaScript (frontend)

App Details

This app allows users to generate images using the DALL路E 3 model by OpenAI. Generated images are saved and can be viewed in the gallery. Users can download any image they create. The app is built with Flask for the backend and uses modern HTML/CSS/JS for the frontend.
